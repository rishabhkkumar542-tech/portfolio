THUMBNAILS FOLDER
=================

This folder is for your THUMBNAIL DESIGNS.

- Put your thumbnail images here (thumbnail1.png, thumbnail2.png, etc.)
- These should be 16:9 aspect ratio (HORIZONTAL - like YouTube thumbnails)
- Examples: 1920x1080, 1280x720
- These are for DISPLAY ONLY (no links attached)

See HOW_TO_ADD_CLIPS_AND_THUMBNAILS.txt for detailed instructions.
