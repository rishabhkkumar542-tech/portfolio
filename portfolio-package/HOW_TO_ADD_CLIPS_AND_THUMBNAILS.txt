========================================
HOW TO ADD CLIPS & THUMBNAILS WITH LINKS
========================================

Your portfolio has two sections:
1. THUMBNAIL DESIGNS (thumbnails/ folder) - WITHOUT links (just for display)
2. CONTENT CLIPS (clips/ folder) - WITH YouTube links (clickable)

This guide covers both!

PART 1: THUMBNAILS (Display Only - No Links)
â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”

FOLDER: thumbnails/
ASPECT RATIO: 16:9 (e.g., 1920x1080, 1280x720)
PURPOSE: Showcase your thumbnail designs

HOW TO ADD:
-----------

STEP 1: Prepare Your Image
- Export your thumbnail as PNG or JPG
- Name it: thumbnail5.png, thumbnail6.png, etc.
- Aspect ratio: 16:9 (horizontal)
- Put it in the "thumbnails" folder

STEP 2: Add to HTML
- Open index.html in a text editor
- Find: <!-- First set -->
- BEFORE the "<!-- Duplicate set -->" comment, add:

                <div class="thumbnail-item">
                    <img src="thumbnails/thumbnail5.png" alt="Description">
                    <div class="thumbnail-overlay">
                        <h3>Your Thumbnail Title</h3>
                        <p>Professional Thumbnail</p>
                    </div>
                </div>

STEP 3: Duplicate for Loop
- Find: <!-- Duplicate set for seamless loop -->
- Add the SAME code again (so slider loops smoothly)

PART 2: CONTENT CLIPS (WITH YouTube Links)
â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”

FOLDER: clips/
ASPECT RATIO: 9:16 (e.g., 1080x1920 - vertical like Instagram/TikTok)
PURPOSE: Link to your YouTube videos/shorts

HOW TO ADD:
-----------

STEP 1: Prepare Your Clip Image
- Export your clip thumbnail as PNG or JPG
- Name it: clip5.png, clip6.png, etc.
- Aspect ratio: 9:16 (VERTICAL)
- Put it in the "clips" folder

STEP 2: Get Your YouTube Video Link
- Go to your YouTube video
- Click "Share" button
- Copy the link (e.g., https://www.youtube.com/watch?v=dQw4w9WgXcQ)
- Note the VIDEO_ID (the part after "v=")
  Example: dQw4w9WgXcQ

For YouTube Shorts:
- Link looks like: https://www.youtube.com/shorts/ABC123
- The VIDEO_ID is: ABC123

STEP 3: Add to HTML
- Open index.html in a text editor
- Find the CONTENT CLIPS section
- BEFORE the "<!-- Duplicate set -->" comment, add:

                <a href="https://www.youtube.com/watch?v=YOUR_VIDEO_ID" target="_blank" class="clip-item">
                    <img src="clips/clip5.png" alt="Clip 5">
                    <div class="clip-overlay">
                        <h3>Your Clip Title</h3>
                        <p>Click to watch</p>
                    </div>
                </a>

STEP 4: Replace YOUR_VIDEO_ID
Replace "YOUR_VIDEO_ID" with your actual video ID:

BEFORE: https://www.youtube.com/watch?v=YOUR_VIDEO_ID
AFTER:  https://www.youtube.com/watch?v=dQw4w9WgXcQ

STEP 5: Duplicate for Loop
- Find: <!-- Duplicate set for seamless loop --> (in clips section)
- Add the SAME code again
